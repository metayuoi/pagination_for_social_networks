:: ИНСТРУКЦИЯ ПО ЗАПУСКУ::

1. Открыть файл vk_favs.js, скопировать всё (Ctrl+A -> Ctrl+C).
2. Находясь на странице https://vk.com/feed?section=likes, открыть браузерную консоль (комбинация клавиш Ctrl+Shift+I).
3. Вставить в консоль скопированный код (Ctrl+V), нажать Enter и дождаться окончания загрузки.
