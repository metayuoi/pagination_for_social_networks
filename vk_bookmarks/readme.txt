:: ИНСТРУКЦИЯ ПО ЗАПУСКУ::

1. Открыть файл vk_bookmarks.js, скопировать всё (Ctrl+A -> Ctrl+C).
2. Находясь на странице https://vk.com/bookmarks, открыть браузерную консоль (комбинация клавиш Ctrl+Shift+I).
3. Вставить в консоль скопированный код (Ctrl+V), нажать Enter и дождаться окончания загрузки.
