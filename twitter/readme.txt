:: ИНСТРУКЦИЯ ПО ЗАПУСКУ::

1. Перейдите на страницу интересующего вас пользователя. Например: https://twitter.com/castalia_alba
2. Откройте браузерную консоль (комбинация клавиш Ctrl + Shift + I).
3. Откройте twitter.js, нажмите Ctrl+A -> Ctrl+C (выделить всё -> скопировать).
4. В браузерную консоль вставьте скопированный код (Ctrl+V), подождите. Загрузка может длиться около минуты, если твитов очень много.
5. Enjoy!
